Assignment 2 Part 2.6.3 - Additional Requests

  • /currentGrade? -- calculates a student's current grade in SER321 based on the grade weights of assignments, quizzes, and the final exam. The user is to input at the minimum the point total earn on assignments and quizzes, each as a separate argument. Additionally, an the point total earned on an exam can be included as a third argument.
    • Example: http://3.133.142.250:9000/currentGrade?assign=89.5&quiz=49.5&exam=258
  • /cashier? -- calculates the change that is to be returned to the customer with the number of each type of coin that should be distributed. The user is to provide a price and a paid argument with appropriate values.
    • Example: http://3.133.142.250:9000/cashier?price=22.23&paid=23.00
